# Исследование: Сценарий Б — шорт #2 монеты

## Хронология исследования

### 1. Первоначальная идея
**Сценарий Б:** монета впервые стала #1 по росту среди USDT perp → откатилась на #2 → вход.

Изначально тестировали **лонг** #2 (ожидание продолжения роста).

### 2. Данные
- **Источник:** Bybit public trade data (`public.bybit.com/trading/`)
- **Период:** 2026-01-04 → 2026-07-02 (180 дней)
- **Пары:** 416 USDT linear perp (исключая heavy: BTC/ETH/SOL и др.)
- **Гранулярность:** 1-минутные свечи из тиковых данных
- **Формат:** `~/bybit_ticks/{SYMBOL}{YYYY-MM-DD}.csv.gz` (74,880 файлов)
- **Кэш:** `/tmp/b1m_days/{date}.pkl` — предагрегированные 1m свечи

### 3. Ошибки найденные и исправленные

#### Ошибка #1: Дубликаты входов
**Симптом:** 1876 входов, 58 входов на CLOUSDT, 17% повторов <1 часа.

**Причина:** `first1 = set()` — "была #1 хоть раз сегодня". Монета болтается #1→#2→#1→#2 и каждый откат на #2 = вход.

**Фикс:** после входа монета попадает в `traded_today = set()` → бан до конца дня.

**Результат:** 1876 → 923 входа (в 2× меньше, но каждый по правилам).

#### Ошибка #2: Формат Bybit WS
**Симптом:** WS подключается, подписки отправлены, но данные не приходят.

**Причина:** `kline.1m.{symbol}` — неверный формат. Bybit V5 использует `kline.1.{symbol}` (интервал без 'm').

**Фикс:** `kline.1m` → `kline.1`. Найдено в коде RDNX бота (`btc_monitor.py`).

### 4. MFE/MAE анализ (923 входа, fixed логика)

**MFE** = Maximum Favorable Excursion (движение в нашу сторону)
**MAE** = Maximum Adverse Excursion (движение против нас)

#### Лонг (рост = профит)

|---|---|---|---|
| 15 мин | 0.94% | 1.03% | MAE > MFE |
| EOD | 5.28% | 6.63% | **MAE > MFE — лонг убыточен** |

**Giveback (EOD): 144%** — монета откатывает ниже точки входа.

#### Шорт (падение = профит)
| Горизонт | MFE med (упала) | MAE med (выросла) | Вывод |

| 15 мин | 1.03% | 0.94% | **MFE > MAE** |
| EOD | 6.63% | 5.28% | **MFE > MAE — шорт прибыльнее** |

**Giveback (EOD): 58%** — шорт держит движение лучше лонга.

**EOD hold:** шорт +1526% Net vs лонг -1628% Net.

### 6. Position sizing

MaxDD -47% был при ставке 100% депозита на сделку.

С нормальным риском:
- 1% риск → MaxDD -1.6%
- **2% риск → MaxDD -3.1%** (базовый)
- 3% риск → MaxDD -4.7%

**Макс серия убытков = 2** (всего 9 стоп-аутов из 923 сделок).
